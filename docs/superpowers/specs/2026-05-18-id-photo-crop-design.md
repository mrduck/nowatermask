# 证件照裁剪+排版工具 设计文档

> 日期: 2026-05-18
> 状态: 已确认

## 1. 概述

在现有去水印工具基础上，新增一个独立的「证件照裁剪」工具。支持手动框选和自动等比例裁剪两种模式，输出单张电子版和6寸相纸排版图。

## 2. 核心需求

### 2.1 定位
- 与去水印工具**完全独立**，各自独立流程
- 两个入口（Tab切换或首页导航）

### 2.2 裁剪模式

| 模式 | 操作方式 | 说明 |
|---|---|---|
| 自动模式 | 选择规格 → 自动居中剪出目标比例 → 可拖拽微调位置 | 适合头像居中、背景干净的照片 |
| 手动模式 | 自用拖拽矩形框自由选区 | 适合特殊构图、多人照片 |

### 2.3 输出
- **单张下载**：标准尺寸电子版 PNG
- **排版下载**：6寸相纸（152×102mm @ 300DPI）自动排版 PNG

## 3. 证件照规格预设

| 规格 | 像素 (300DPI) | 宽高比 | 物理尺寸 | 6寸排版 |
|---|---|---|---|---|
| 1寸 | 295×413 | ~1:1.40 | 25×35mm | 2行4列 = 8张 |
| 小1寸 | 260×378 | ~1:1.45 | 22×32mm | 2行4列 = 8张 |
| 2寸 | 413×579 | ~1:1.40 | 35×49mm | 2行2列 = 4张 |
| 小2寸 | 413×531 | ~1:1.29 | 35×45mm | 2行2列 = 4张 |
| 大一寸 | 390×567 | ~1:1.45 | 33×48mm | 2行3列 = 6张 |

## 4. 技术架构

### 4.1 整体架构

```
前端 Canvas 交互层              后端 cv2 精确处理层
┌──────────────────┐           ┌─────────────────────┐
│ 上传照片            │           │ POST /api/crop       │
│ 规格选择            │ ──坐标──→ │ cv2 精确裁剪+缩放     │
│ 裁剪框拖拽 (Canvas)  │ ←──图片── │ 300DPI 高清输出      │
│ 实时预览            │           │                     │
│ 排版视图切换         │ ──请求──→ │ POST /api/layout     │
│ 下载单张/排版图      │ ←──图片── │ 6寸相纸自动排版      │
└──────────────────┘           └─────────────────────┘
```

### 4.2 前端组件

| 组件 | 职责 |
|---|---|
| `CropPage.vue` | 证件照裁剪主页面，包含规格选择、模式切换、下载 |
| `CropCanvas.vue` | Canvas裁剪框交互（拖拽/缩放/锁定比例） |
| `LayoutPreview.vue` | 6寸排版预览视图 |

### 4.3 后端 API

| 端点 | 方法 | 入参 | 出参 | 说明 |
|---|---|---|---|---|
| `/api/crop` | POST | `file`, `x`, `y`, `w`, `h`, `target_w`, `target_h` | 裁剪后PNG | 先按坐标裁剪，再缩放到目标尺寸 |
| `/api/layout` | POST | `file`, `spec` (如 "1inch") | 6寸排版PNG | 自动排版到6寸相纸 |

### 4.4 技术选型

| 层 | 技术 | 理由 |
|---|---|---|
| 前端裁剪框交互 | Vue3 + Canvas API | 已有依赖，纯前端即可实现拖拽+比例锁 |
| 后端精确裁剪 | cv2 (已安装) | 无需新依赖，cv2 裁剪+缩放成熟可靠 |
| 排版计算 | cv2 `np.tile` 或逐行拼接 | 纯数组操作，简单可靠 |

## 5. 前端交互流程

### 5.1 页面状态机

```
idle → crop → done
  ↑               │
  └─── reset ─────┘
```

### 5.2 自动模式流程

```
1. 上传照片
2. 选择证件照规格（1寸/2寸/...）
3. Canvas 显示锁定比例的裁剪框（自动居中）
4. 可拖拽移动裁剪框位置
5. 可缩放裁剪框（锁定比例）
6. 点击裁剪 → POST /api/crop
7. 预览结果 → 下载单张 | 查看排版
```

### 5.3 手动模式流程

```
1. 上传照片
2. Canvas 显示原图
3. 自由拖拽矩形框选区（不锁定比例）
4. 点击裁剪 → POST /api/crop（按实际选区裁剪，不缩放）
5. 预览结果 → 下载
```

### 5.4 Canvas 裁剪框交互规则

- 拖拽四角：缩放（自动模式锁定比例，手动模式自由缩放）
- 拖拽内部：移动裁剪框
- 裁剪框不能超出图片边界
- 实时显示裁剪后像素尺寸

## 6. 排版本设计

### 6.1 6寸相纸规格
- 尺寸: 1800×1200 px (300 DPI = 152×102mm)
- 背景色: 白色 (#FFFFFF)
- 照片间距: 10px
- 边距: 10px

### 6.2 排版计算

```
6寸画布 1800×1200 px

1寸排版: 2行4列
  每张: 295×413 px
  列位置: 10 + i*(295+10), i=0,1,2,3
  行位置: 10 + j*(413+10), j=0,1

2寸排版: 2行2列
  每张: 413×579 px
  列位置: 10 + i*(413+10), i=0,1
  行位置: 10, 10+579+10
```

## 8. 非功能需求

- 裁剪框操作流畅（60fps Canvas渲染）
- 后端裁剪+缩放 < 500ms
- 排版生成 < 1s
- 输出图片保持原始画质
- 移动端适配（触摸拖拽）
